# 👋 About Me

I am **Sleepy**, a software developer from Romania focused on Minecraft development, desktop applications, and modern web experiences.

I enjoy building high quality plugins, Windows applications, moderation tools, and custom systems while constantly learning new technologies and improving existing projects.

* Developing custom Minecraft plugins like SingleKit
* Building Minecraft screensharing and moderation tools including JarInspect
* Creating Windows desktop applications with C#
* Developing modern web applications
* Passionate about performance, stability, and clean software
